# DSL Design System

Дизайн-система Larixon Mobile. Источник правды для дизайна и разработки — спеки в папке `docs/` (оглавление: **docs/README.md**).

## Структура

- **`docs/`** — спеки токенов, типографики, цветов и компонентов (Markdown).
- **`figma-plugin/ds-audit-scanner/`** — Figma-плагин для аудита покрытия токенами.
- **`scripts/figma-audit-scanner.js`** — скрипт того же аудита для запуска через Figma MCP.
- **`static-site/`** — отдельный статический сайт-витрина (раздаётся независимо).
