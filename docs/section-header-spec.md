# Section Header

## §1 Обзор

Section Header — заголовок секции контента с опциональной кнопкой-действием справа. Используется внутри фильтров, форм, Bottom Sheet, списков с группировкой.

```
Статус                          ↺ Сбросить
Price
```

---

## §2 Когда использовать

**Использовать:**
- Заголовок логической секции внутри Sheet / формы / экрана.
- Когда секцию нужно именовать и / или дать быстрое действие (сброс, выбрать всё, ещё).

**Не использовать:**
- Как заголовок всего экрана — для этого Top App Bar v2.
- Как заголовок карточки — там своя типографика.

---

## §3 Anatomy

```
┌──────────────────────────────────────────┐
│ Статус                    [icon] Сбросить│
└──────────────────────────────────────────┘
  ↑ title (FILL)             ↑ action (HUG, optional)
```

**Слои:**
- `title` — основной заголовок секции
- `action` (boolean) — ссылочная кнопка справа: `[icon 16dp] [label]`

---

## §4 Варианты

| Вариант | Описание |
|---|---|
| `Action=False` | Только заголовок |
| `Action=True` | Заголовок + действие справа |

---

## §5 Токены

### Title

| Параметр | Токен | Значение |
|---|---|---|
| Типографика | `Heading/H4 Bold` | 18/24, Inter Bold |
| Цвет | `Text&Icon/Primary` | — |

### Action

| Параметр | Токен | Значение |
|---|---|---|
| Иконка | `size/xs` placeholder | 16 dp |
| Типографика | `Base/Body 2` | 14/20, Inter Regular |
| Цвет (текст + иконка) | `Accent/Link` | — |
| Gap иконка↔текст | — | 4 dp |

### Layout

| Параметр | Значение |
|---|---|
| Горизонталь | SPACE_BETWEEN (title FILL, action HUG) |
| Высота | HUG (= lineHeight title = 24 dp) |
| Ширина | FILL в контексте (мастер 360 dp) |
| Padding | 0 — отступы на стороне родителя через `section/*` spacing |

> Иконка в `action` — placeholder 16×16. Заменяй через INSTANCE_SWAP на нужную иконку (обычно `ic_refresh`, `ic_close` или контекстная).

---

## §6 Позиционирование

Section Header — inline-элемент. Spacing выше и ниже задаётся родительским auto-layout через `section/padding-*` или `stack/*` токены. Компонент сам по себе без padding'ов.

---

## §7 Доступность

- `title` — heading-роль (`accessibilityTraits: .header` iOS / `heading` Android).
- `action` — отдельная tappable-область. Минимальный tappable размер 44dp обеспечивает родитель через padding секции.
