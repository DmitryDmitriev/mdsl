# Как открыть дизайн-систему

## Вариант 1: Статичный сайт (проще всего)

В проекте есть папка **`static-site`** с одним файлом `index.html`.

**Как открыть локально в браузере:**

1. В Finder открой папку проекта и зайди в `static-site`.
2. Дважды кликни по **`index.html`** — он откроется в браузере.

Или из терминала:
```bash
open /Users/fondmitry/Yandex.Disk.localized/_Cursor_/DSL/static-site/index.html
```

Никакой сервер не нужен: всё в одном HTML-файле, работает по двойному клику.

---

## Вариант 2: Storybook (полная версия)

Делай **по шагам в терминале** (в Cursor: меню Terminal → New Terminal).

---

## Шаг 1. Перейти в папку проекта

Путь должен быть с подчёркиваниями: **`_Cursor_`**, не `Cursor`.

```bash
cd /Users/fondmitry/Yandex.Disk.localized/_Cursor_/DSL
```

---

## Шаг 2. Установить зависимости (один раз)

Если ещё не ставил:

```bash
npm install
```

---

## Шаг 3. Собрать Storybook

```bash
npm run build-storybook
```

Дождись окончания (в конце будет строка про `storybook-static`).

---

## Шаг 4. Запустить просмотр

```bash
npm run serve-storybook
```

В терминале появится что-то вроде:

```
   Serving!
   - Local: http://localhost:5000
```

Открывай в браузере **именно этот адрес** (если порт другой — тот, что показан). Не открывай `http://localhost:3000`, если в выводе указан другой порт.

---

## Шаг 5. Открыть в браузере

Скопируй адрес из терминала (например `http://localhost:5000`) и вставь в адресную строку браузера.

---

## Если не работает

- **«No such file or directory»** — проверь путь: везде должно быть `_Cursor_` (с подчёркиваниями).
- **«Port already in use»** — в терминале будет указан другой порт (например 5001), открой его.
- **Пустая страница или 404** — не открывай `index.html` через «Открыть файл». Обязательно используй команду `npm run serve-storybook` и адрес из терминала.

## Вариант без npm (если есть Python)

После шагов 1–3:

```bash
cd storybook-static
python3 -m http.server 5000
```

Потом открой в браузере: **http://localhost:5000**
